# KUCCPS Course Calculator

A web application that helps Kenyan students find courses they qualify for based on their KCSE grades.

## Features

- Enter KCSE grades and find eligible courses
- Degree, Diploma, Certificate, and KMTC programs
- Payment via Paystack (KES 200)
- Email results after payment
- User dashboard with statistics
